# 小米音箱

A MiMusic JS plugin generated by `create-mimusic-plugin`.

## Development

```bash
npm install
npm run dev         # watch + auto-upload to local MiMusic
npm run build       # produce dist/xiaomi.jsplugin.zip
npm run validate    # verify plugin.json hashes
```

## Description

小米音箱播放歌曲
